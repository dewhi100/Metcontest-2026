For SMART users:
If you're starting a new project:
-Apply the ips before loading the ROM, this way smart will save the GFX/palette changes
-add the ASM to the ASM folder and use it

If you already have a busy project:
-Make a temporary new project
-Aply ips patch to the temporary project
-Open temporary project and current project at once
-Copy from temporary to current one (the gfx and palettes of the pause screen)
