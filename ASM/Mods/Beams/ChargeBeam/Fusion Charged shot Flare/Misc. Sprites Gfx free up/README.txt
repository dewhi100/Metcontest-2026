Miscellaneous  graphics freeing, by Tundain

This patch frees up a grand total of 28 8x8 tiles in the misc. sprites that you can now use to make whatever fancy stuff you want.
However, it does come with a few side effects, but hopefully none too noteworthy.
-Powerbombs don't animate quicker in their last few frames anymore, now it's at a constant rate
-All big health drops animate in sync
-Bomb explosions and plasma sba now use a different explosion animation, using the current beam palette.
-Elevators now require you to add them in the enemy types list in order to display properly.


Info on how to enable usage of the elevator enemy in the editor with SMART/SMILE is in the respective folder.
